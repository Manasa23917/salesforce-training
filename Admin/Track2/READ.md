# Salesforce Administrator Track – Day 2
## Reports & Dashboards

### What is a Report?
A Salesforce Report is a list of records that helps users analyze and organize data. Reports are used to track performance, identify trends, and support business decisions.

### What is a Dashboard?
A Dashboard is a visual representation of report data using charts, graphs, gauges, and metrics. It helps management quickly understand business performance.

### Difference Between Reports and Dashboards

| Report | Dashboard |
|----------|----------|
| Shows detailed data | Shows visual summary |
| Used for analysis | Used for monitoring |
| Can exist independently | Uses reports as data source |
| Tabular format | Charts and graphs |

---

## Five Important College Reports

### 1. Student Attendance Report
- Data: Student name, attendance %
- Viewed By: Faculty, HOD
- Refresh: Daily

### 2. Placement Status Report
- Data: Student, company, placement status
- Viewed By: Placement Officer, Principal
- Refresh: Weekly

### 3. Faculty Workload Report
- Data: Faculty, courses assigned
- Viewed By: HOD, Principal
- Refresh: Monthly

### 4. Internship Tracking Report
- Data: Student, company, internship status
- Viewed By: Placement Officer
- Refresh: Weekly

### 5. Student Performance Report
- Data: GPA, marks, department
- Viewed By: Faculty, Principal
- Refresh: Monthly

## Interview Questions

### What is a Salesforce Report?
A report is a collection of records used for analysis.

### What is a Dashboard?
A dashboard is a visual display of report data.

### What is a Dynamic Dashboard?
A dashboard that displays data based on the logged-in user.

### What are KPIs?
Key Performance Indicators used to measure performance.

### Why are dashboards important?
They provide quick insights for management decision-making.
