# Salesforce Administrator Track – Day 5
## Lightning Experience & Lightning App Builder

### What is Lightning Experience?
Lightning Experience is Salesforce's modern user interface that provides an improved and productive experience through customizable pages, dashboards, and apps.

### What is Lightning App Builder?
Lightning App Builder is a drag-and-drop tool that allows administrators to create and customize pages without writing code.

It is used to build:
- Home Pages
- Record Pages
- App Pages

## How Lightning App Builder Improves Productivity

1. Removes unnecessary fields.
2. Simplifies page layouts.
3. Improves navigation.
4. Shows relevant information only.
5. Reduces user clicks.
6. Creates role-specific pages.
7. Displays important reports on pages.
8. Uses dynamic visibility for personalization.
9. Improves data entry speed.
10. Enhances overall user experience.

## Reflection

### Why Should Different Users Have Different Interfaces?

Different users perform different tasks. A Student, Faculty member, Placement Officer, and Administrator require different information and tools. Customized interfaces improve productivity, reduce confusion, and provide a better user experience.

## Interview Questions

### What is Lightning Experience?
The modern Salesforce user interface designed for productivity and customization.

### What is Lightning App Builder?
A drag-and-drop tool used to create and customize Lightning pages.

### Difference Between Home Page and Record Page?
- Home Page: Displays user-specific information and dashboards.
- Record Page: Displays details of a specific record.

### What is an App Page?
A custom page that can be added to a Lightning App.

### Why Customize Page Layouts?
To show relevant information and improve user productivity.

### What Are Lightning Components?
Reusable UI elements used to build Lightning pages.

### What is Dynamic Visibility?
A feature that displays components based on user criteria or record data.

### How Can Admins Improve Productivity Without Coding?
By using page layouts, Lightning App Builder, dynamic forms, dashboards, and automation tools.

### What is a Lightning App?
A collection of tabs, objects, pages, and features designed for a specific business process.
