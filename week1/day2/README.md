# Day 2 - Salesforce Platform Basics

## What is Salesforce Platform?

Salesforce Platform is a cloud-based platform used to manage customer data, business processes, and applications. It helps companies build and customize business systems easily.

---

## What is an App?

An App in Salesforce is a collection of tabs, objects, and tools used for specific business purposes.

Example:
Sales App

---

## What is an Object?

An Object is used to store data in Salesforce.

Examples:
Account, Contact, Opportunity

---

## What is a Tab?

A Tab is used to open and access objects or features in Salesforce.

Examples:
Accounts Tab, Contacts Tab

---

## How CRM Concepts Fit into Salesforce Platform

CRM concepts like Account, Contact, and Opportunity are stored as objects inside Salesforce apps. Users access them through tabs and manage customer data easily.

---

## Difference Between Configuration and Coding

### Configuration (No Code)

Configuration means customizing Salesforce using clicks and settings without programming.

Examples:
- Creating fields
- Creating reports

### Coding (Apex)

Coding means using programming to build advanced functionality in Salesforce.

Examples:
- Custom business logic
- API integrations

Multi-tenant architecture means multiple companies use the same Salesforce platform securely while keeping their data separate.
